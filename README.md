# Omniauth::Smarthr

This is omniauth strategy for authenticating to your SmartHR service used OAuth 2.0 style.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'omniauth-smarthr', github: 'SonicGarden/omniauth-smarthr'
```
